La página web fue desarrollada por completa con el editor NetBeans  y escrita en los lenguajes de programación HTML, PHP y MYSQL.

* Para poder buscar un producto primero debe ingresar el nombre del producto y después presionar el botón buscar, 
  si el nombre no está registrado o esta mal escrito la página mostrara todos los registros. 

* Para buscar por categoría primero se debe seleccionar una categoría y después presionar el botón buscar, si busca por nombre 
al vez que se busca por categoría, imperar la buscada por nombre.

* Hasta el momento la página no presento errores, busca por código identificador y filtra los resultados por categoría. Además de presentar 
  todos los resultados en una sola página.


Se implementaron las mejoras:

La estructura interna del código está más ordenada, además de tener comentarios explicativos de cada función.

APIREST: ahora la aplicación es una APIREST porque interactúa mediante el método GET con el servidor en línea,
la conexión fue realizada con el lenguaje PHP.

FRONT-END: el FRONT-END fue desarrollado casi por completo con HTML, pero incluye el paquete gratuito de bootraps y un css (myowncss) propia para adaptar mejor la visualización de los productos.
